# vue-lyc-demo

基于vue1.0的vue-cli改造的vue多页面应用脚手架，并集成jquery库。

## 构建步骤

		``` bash
		# 安装依赖
		npm install
		# 本地测试
		npm run dev
		# 打包
		npm run build
		
		```
		
在本地调试启动后访问：[index(http://localhost:8077)](http://localhost:8077) | [about(http://localhost:8077/about.html)](http://localhost:8077/about.html) 即可
